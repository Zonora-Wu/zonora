# 博客文章模板

博客文章统一存放在 `data/blog`。文件名会直接成为 URL slug，例如：

```text
data/blog/my-new-post.md
→ /blog/my-new-post/
```

每篇文章必须使用以下结构：

````md
---
title: 文章标题
date: 2026-06-21
excerpt: 显示在博客列表和搜索结果中的摘要。
---

# 文章标题

正文内容。

## 一级章节

章节正文。

### 二级章节

章节正文。

```tsx
const example = "代码块会自动带语言标识和复制按钮";
```
````

保存文件后系统会自动完成：

- 加入博客列表并按日期倒序排列；
- 生成文章静态路由；
- 应用统一文章标题、正文和左侧章节导航布局；
- 根据 `##`、`###` 标题生成目录；
- 为代码块添加语言标签与一键复制按钮；
- 使用标题和摘要生成页面 SEO 元数据。
